# 脚本目录

这里用于存放 `Innocence` 项目的开发辅助脚本。

## 当前状态

当前目录还没有正式脚本文件，只有这份说明文档。

这意味着：

- 目前大多数开发动作还是通过手动命令完成
- 后续如果需要稳定复用，建议把常用流程沉淀成脚本

## 适合放在这里的脚本

- 本地一键启动脚本
- 前后端联调启动脚本
- Windows 发布打包脚本
- 数据库初始化脚本
- 文档生成或校验脚本
- Git 提交前检查脚本

## 建议优先补充

1. Flutter Windows 发布构建脚本
2. Spring Boot 本地启动脚本
3. Docker 本地依赖启动脚本
4. 前端格式化与静态检查脚本

## 命名建议

为了后续维护更顺手，建议：

- Windows 使用 `.ps1`
- 跨平台辅助脚本可用 `.sh`
- 纯说明性或模板文件可以配套补 `README`
